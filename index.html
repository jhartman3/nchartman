Here, you'll find assignments completed for the course, as well as some miscellaneous information on the designer.

The website will be a work in progress for some time, so browse to your heart's content!

On this page, you will find links to the assignments completed for the course. More will be added as the course progresses.

Hello, World! - A simple Hello World implementation to demonstrate basic HTML knowledge.
Magazine Ad - A test to determine student ability to recreate a page in HTML, with a magazine advertisement as the source page.
CSS Commenting - A given CSS document commented based on knowledge of CSS.
jQuery Experiment - A page used to demonstrate initial knowledge of jQuery.
Database Tutorial - A document detailing the creation of tables and select statements in a database.
Database Inserter - A page that can insert game entries into a database.
